# NewsFresh App
### An Indian News App

# About the App #
### Language Used - Kotlin 
### API used - NewsApI

### User can:

- View a number of top general news (scroll down to get more)
- Watch the details of the news i.e, image, headlines and the author
- Click on the image to view the detailed article to it

### Uses :

- RecyclerView
- Volley
- Glide
